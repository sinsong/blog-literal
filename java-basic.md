# 命名

类名：单词首字母大写。
方法名：小写字母开头，驼峰命名法。
源文件名：源文件名必须和公开类名相同。
主方法如阔：主类的main方法。

# 标识符

以字母，美元符，下划线开始。
剩下的可以有字母，美元符，下划线，数字。
不能是关键字。
大小写敏感。
