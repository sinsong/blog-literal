Bootstrap v3.3.7 v3最后一个版本，最后一个使用LESS的版本。。。

# 目录结构

* less/		CSS
* js/		JS
* fonts/	字体
* dist/		预编译文件

# 编译

##使用Grunt

`npm install -g grunt-cli`
`/bootstrap/ $ npm install`

## 命令

`grunt dist` 重新生成`dist`，编译文件。
`grunt test` 运行测试用例
