```html
<a href="链接地址">链接文字</a>
```

所有浏览器中，链接的默认外观是：
* 未被访问的链接-带下划线的蓝色文字
* 已被访问的链接-带下划线的紫色文字
* 选定的链接-带下划线的红色文字

请让我施展魔法，让超链接变得更漂亮吧！

去掉下划线！
```css
a
{
	text-decoration: none;
}
```

伪类！你的不同状态！

```css
a:link {} /* 未访问的链接 */
a:visited {} /* 已访问的链接 */
a:hover {} /* 鼠标悬停 */
a:active {} /* 选定的链接 */
```

> 提示：在 CSS 定义中，a:hover 必须被置于 a:link 和 a:visited 之后，才是有效的。
> 提示：在 CSS 定义中，a:active 必须被置于 a:hover 之后，才是有效的。
> 提示：伪类名称对大小写不敏感。
